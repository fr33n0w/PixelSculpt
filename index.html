PixelSculpt
Free, open-source lithophane and color lithophane (HueForge-style) generator. Upload an image, configure filament layers with Transmission Depth simulation, preview in 3D with backlight, and export print-ready STL files — all in your browser, no server needed.

Image & Adjustments (Left Panel)

Image Upload
Drag & drop or click to load PNG/JPG/WebP. The image is converted to a grayscale heightmap that drives the 3D surface relief.
Dimensions
Set physical width/height in mm. Lock aspect ratio to auto-calculate height from the image proportions.
Thickness
Min thickness = base plate. Max thickness = tallest point. Layer height controls print resolution and total layer count.
Brightness / Contrast / Gamma
Fine-tune the heightmap. Invert for negative effect, mirror for backlit mounting orientation.

Filament Layers & TD (Right Panel)

AMS / Manual Mode
AMS mode: multi-color auto filament change. Manual mode: pause-at-layer instructions for hand swapping filament.
Transmission Depth (TD)
Each filament has a TD value (mm) — how much light passes through. Lower TD = more opaque. Simulates Beer-Lambert absorption.
Filament Presets
28+ built-in presets (Bambu, Prusament, eSun, Polymaker). Add custom presets, import/export JSON.
STL Export
Binary (smaller) or ASCII STL. Optional border frame. Full watertight mesh with top, bottom and side faces.
